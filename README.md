 ## 10일차
 - 파이썬 응용

   - 실행파일 만들기
     - PyInstaller 모듈 설치
     ```shell
     > pip install pyinstaller
     > pyinstaller -w -F pythonfile.py
     ```
     - -w는 윈도우창만 실행 콘솔창삭제, -F _internal 폴더 생성안되도록, oneFile로 만들어 준다
